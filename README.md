## 📌 Sobre o projeto
 
Página pessoal de mapa de carreira e portfólio desenvolvida como projeto final do programa **PROA/Senac Full-Stack Python**. Apresenta minha trajetória profissional planejada, habilidades técnicas (hard skills) e comportamentais (soft skills), além de um resumo da minha experiência atual no projeto **BusSense**.
 
A página foi construída com foco em:
- Design responsivo para desktop e mobile
- Animações suaves de barra de progresso ativadas por scroll (IntersectionObserver)
- Acessibilidade com atributos ARIA nas barras de habilidade
- Carregamento performático com Bootstrap customizado

## ✨ Funcionalidades
 
- **Mapa de carreira interativo** — Linha do tempo com 4 etapas: Júnior → Pleno → Sênior → Arquiteto de Soluções em Nuvem
- **Barras de progresso animadas** — Ativadas no scroll via `IntersectionObserver`, com transição suave de 1.5s
- **Download de currículo** — Botão direto para baixar o PDF do CV
- **Links sociais** — LinkedIn e GitHub com ícones Font Awesome
- **Layout responsivo** — Adaptado para qualquer tamanho de tela com Bootstrap 5

## 🚀 Como rodar localmente
 
Não é necessário nenhuma dependência ou build. Basta clonar e abrir:
 
```bash
# 1. Clone o repositório
git clone https://github.com/gabriel-r-silvaa/mapa-de-carreira.git
 
# 2. Acesse a pasta
cd mapa-de-carreira
 
# 3. Abra o arquivo no navegador
# No VS Code, use a extensão Live Server
# Ou clique duas vezes no index.html
```
 
> 💡 **Dica:** Use a extensão [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) no VS Code para recarregamento automático ao editar.

## 🌐 Deploy com GitHub Pages
 
Este projeto está hospedado gratuitamente via **GitHub Pages**.
 
Para fazer o deploy do seu fork:
 
1. Vá em **Settings** → **Pages**
2. Em *Source*, selecione **Deploy from a branch**
3. Escolha a branch `main` e a pasta `/ (root)`
4. Clique em **Save**
5. Aguarde ~1 minuto e acesse: `https://<seu-usuario>.github.io/<nome-do-repo>/`
